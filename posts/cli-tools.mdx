Recently I realize I've accumulated quite a few packages and wanted to do some cleaning and organizing.

These are the ones I kept and find _useful._

<Note>Anything you think I might be missing out on? Let me know!</Note>

**Here goes!**

## Must Haves

(that I don't need to think about)

### Utilities

- [fzf](https://github.com/junegunn/fzf) - Command-line fuzzy finder written in Go
- [atuin](https://github.com/atuinsh/atuin) - Improved shell history for zsh, bash, fish and nushell
- [bat](https://github.com/sharkdp/bat) - Clone of cat(1) with syntax highlighting and Git integration
- [clipboard](https://getclipboard.app/) - Cut, copy, and paste anything, anywhere, all from the terminal
  - The easiest way to move files around
- [zoxide](https://github.com/ajeetdsouza/zoxide) - shell extension to navigate your file-system faster
- [yazi](https://yazi-rs.github.io/) - Blazing fast terminal file manager written in Rust, based on async I/O
- [eza](https://github.com/eza-community/eza) - Modern, maintained replacement for ls
- [gum](https://github.com/charmbracelet/gum) - Tool for glamorous shell scripts
- [direnv](https://direnv.net/) - Load/unload environment variables based on $PWD
  - I use this at work to automatically switch between dev environments (correct node/go/python versions etc.)
- [tldr](https://github.com/isacikgoz/tldr) - fast and interactive tldr client written with go
- [serpl](https://github.com/yassinebridi/serpl) - a simple terminal UI for search and replace, ala VS Code.
  - An easier way to find and replace. When I need more control I use [ripgrep](https://github.com/BurntSushi/ripgrep).

### Git and GitHub

- [lazygit](https://github.com/jesseduffield/lazygit?tab=readme-ov-file#configuration) - A simple terminal UI for git commands, written in Go
- [gh](https://github.com/cli/cli) - GitHub command-line tool
- [diff-so-fancy](https://github.com/so-fancy/diff-so-fancy) - Good-lookin' diffs with diff-highlight and more
- [tig](https://jonas.github.io/tig/) - Text interface for Git repositories
- [svu](https://github.com/caarlos0/svu) - Semantic Version Utility
  - I use this when releasing my CLI tool to bump versions

### Process Management

- [tmux](https://github.com/tmux/tmux) - terminal multiplexer
- [sesh](https://github.com/joshmedeski/sesh) - Smart terminal session manager for tmux
- [mprocs](https://github.com/pvolok/mprocs) - Run multiple commands in parallel
- [lazydocker](https://github.com/jesseduffield/lazydocker) - Lazier way to manage everything docker
  - I use this when developing my TUI and find broken icons
- [k9s](https://k9scli.io/) - kubernetes CLI to manage your clusters in style!
- [bpytop](https://github.com/aristocratos/bpytop) - Linux/OSX/FreeBSD resource monitor

### Disk Management

- [dua-cli ](https://github.com/Byron/dua-cli)- view disk space usage and delete unwanted data, fast
  - Run `dua i` for interactive mode
- [dust](https://github.com/bootandy/dust) - more intuitive version of du in rust
- [kondo](https://github.com/tbillington/kondo) - save disk space by cleaning non-essential files from software projects

### Golang
